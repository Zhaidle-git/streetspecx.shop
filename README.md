# streetspecx.shop
responsive business website

**1. Project Overview**
STREETSPECX is a black, blue and gold themed t-shirt business website designed to showcase streetwear products in a modern and minimal style. The website focuses on clean layout,responsive design, and easy navigation for users.
Live site: https://zhaidle-git.github.io/streetspecx.shop/

**2. Project Goals**
  • Create a stylish and modern online presence for a streetwear brand
  • Showcase products clearly using a minimal black, blue and gold theme
  • Ensure the website is responsive (works on mobile and desktop)
  • Practice using HTML and CSS and some javascript for real-world web design
  • Deploy the website using GitHub Pages
  
**3. Technologies Used**
- HTML5 – Used for structuring the website (header, sections, footer)
- CSS3 – Used for styling, layout, and responsiveness (Flexbox/Grid)
- Internal Javascript - to create interactive and dynamic web content
- Font Awesome – Used for social media icons
- GitHub Pages – Used to host and publish the website online

**4. Features**
  • Clean black, blue and gold aesthetic design
  • Responsive layout for different screen sizes
  • Navigation bar for easy access
  • Social media icons
  • Product showcase sections
  • Footer with branding and links
  
**5. How to Run the Project Locally**
Step 1: Download the project using Git:git clone https://github.com/zhaidle-git/streetspecx.com.git
Step 2: Open the project folder and locate index.html
Step 3: Open index.html in a web browser (double-click or open with browser)

**6. Deployment**
The website is hosted using GitHub Pages. Steps include pushing files to a repository, enablingPages in settings, selecting the branch, and accessing the generated live link.
